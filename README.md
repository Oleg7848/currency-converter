# Currency Converter (Конвертер валют)

## Автор
**Олег Максимов**

## Описание
GUI-приложение для конвертации валют, созданное с использованием **только встроенных модулей Python**. 
Не требует установки дополнительных библиотек!

Приложение поддерживает:
- Конвертацию между 15 популярными валютами
- Два режима работы: через API и демо-режим
- Сохранение истории конвертаций в JSON
- Валидацию пользовательского ввода
- Экспорт истории в отдельный файл

## Используемые встроенные модули
- `tkinter` - графический интерфейс
- `urllib` - HTTP запросы к API (не требует requests!)
- `json` - работа с JSON файлами
- `datetime` - временные метки
- `ssl` - безопасные соединения
- `os` - работа с файловой системой
- `typing` - аннотации типов

## Как получить API ключ (опционально)

Хотя приложение работает без API ключа в демо-режиме, для получения актуальных курсов:

1. Перейдите на [exchangerate-api.com](https://www.exchangerate-api.com/)
2. Нажмите "Get Free Key"
3. Зарегистрируйтесь с email
4. Получите бесплатный ключ
5. Вставьте ключ в переменную `self.api_key` в `main.py`
6. Перезапустите приложение

**Лимиты бесплатного тарифа:**
- 1500 запросов/месяц
- Обновление курсов: 24 часа
